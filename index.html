Teaching

My teaching connects mathematical foundations, modern AI systems, and real-world applications. I design graduate-level courses that emphasize conceptual depth, hands-on implementation, reproducible experimentation, and responsible use of AI.

DATA 220: Mathematical Methods for Data Analytics
Linear algebra, optimization, probability, statistics, and mathematical reasoning for data science.
DATA 245: Machine Learning Technologies
Supervised and unsupervised learning, feature engineering, dimensionality reduction, model evaluation, explainability, and applied machine learning systems.
DATA 255: Deep Learning Technologies
Neural networks, CNNs, RNNs, transformers, generative models, and applied deep learning projects.
DATA 266: Generative AI and Language Models
LLMs, prompt engineering, RAG, fine-tuning, RLHF, multimodal AI, and responsible deployment.

Teaching Approach

  • Project-based learning with real-world AI problems.
  • Transparent assignment design using Purpose–Task–Criteria structure.
  • Emphasis on reproducibility, model evaluation, and error analysis.
  • Human-AI collaboration with critical reflection rather than blind prompting.
  • Industry-relevant skills including Python, deep learning frameworks, LLM workflows, and MLOps thinking.
