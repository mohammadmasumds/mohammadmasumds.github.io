Multimodal Recommendation Systems

Vision Transformers • Large Language Models • RAG • Attribute-Aware Product Understanding

This research direction develops multimodal AI systems that improve product search, recommendation, and information management by combining visual understanding, language reasoning, and retrieval-augmented generation. The central goal is to move beyond keyword-based retrieval and image-only similarity toward systems that better understand customer intent, product attributes, and visual context.

Core Contributions

  • Integrates Vision Transformers and LLMs to improve e-commerce search and recommendation quality.
  • Uses LLM-generated captions to reduce the impact of noisy product descriptions and keyword stuffing.
  • Combines visual and textual representations through multimodal fusion for more context-aware retrieval.
  • Explores RAG-based query enhancement to better align recommendations with user intent.
  • Develops attribute-aware representation learning for fine-grained product understanding.

Representative Projects and Papers

BiLens: Multimodal Recommendation Framework

BiLens combines Vision Transformers and LLM-generated captions to improve product recommendation accuracy. Visual features are extracted using ViT, while image captions are transformed into contextual word embeddings. These representations are integrated through early fusion and compared using cosine similarity to retrieve products that better reflect customer intent.

VisioRAG: Vision Transformers and RAG for Recommendation

VisioRAG extends multimodal recommendation by using Retrieval-Augmented Generation and LLM-based caption enhancement. The framework uses Florence-2-large for image captioning, BERT for contextual embeddings, and Gemini for caption enhancement, enabling richer product representations and more human-like query understanding.

FACTOR: Constrained Representation Refinement

FACTOR addresses fine-grained attribute recognition in fashion and e-commerce by improving structured attribute prediction while preserving the stability of pretrained vision-language models. The framework focuses on attributes such as color, material, and pattern, which are essential for interpretable product understanding.

Impact

Together, these projects support the design of next-generation recommendation systems that understand products more like humans do: through visual details, semantic attributes, contextual descriptions, and user intent. This direction has applications in e-commerce, digital advertising, visual search, and intelligent information management.
